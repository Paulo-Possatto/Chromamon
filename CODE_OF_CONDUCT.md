# Code of Conduct for Collaborators
### Our commitment
We, as collaborators and supporters of this project, are committed to promoting an open, welcoming and inclusive environment. Our aim is to create a space where everyone can contribute in a constructive and respectful way.

### Expected behaviors
- Show empathy and respect for others.
- Communicate clearly and politely.
- Accept constructive criticism with professionalism.
- Collaborate in a spirit of good faith and honesty.

### Unacceptable behavior
- Using discriminatory, abusive or offensive language.
- Personal attacks or provocations.
- Harassment of any kind.
- Behavior that harms the well-being of the community.


### Reporting Violations
If you witness or are the victim of a violation of the code of conduct, please add a [Issue](https://github.com/Paulo-Possatto/Chromamon/issues) with the "Report Violation" format.

### Consequences
Violations of the code of conduct can result in:

- Formal warnings.
- Temporary suspension.
- Permanent ban, depending on the severity of the incident.

### Scope
This code of conduct applies to all interactions related to the project, whether they take place in the GitHub repository, on social networks or in other communication channels associated with the project.

### Inspiration
This code of conduct was inspired by the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
